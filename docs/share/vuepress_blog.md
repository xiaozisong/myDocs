# Vuepress2 从 0 到 1 搭建自己的博客文档

VuePress 是一个以 Markdown 为中心的静态网站生成器。我们可以使用 Markdown 来书写内容（如文档、博客等），然后 VuePress 会生成一个静态网站来展示它们。

本文的内容是我在使用 `vuepress 2.x` 搭建自己的博客文档的一些经验。
